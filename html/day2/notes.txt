day2 html notes:

styling tags:
    to bold text: <b>, <strong>
    to italic text: 
        - <i>
        - <em>
    to underline: <u>
    to delete or strike through: <del>
    to make font size small: <small>
    to highlight word: <mark>

nested tags: tags inside tags (outer tags and inner tags)


horizontal line: <hr>


extension: live server (to see your changes live in browser, just save the file)
    - no need to reload or refresh your page

 DOM: Document object model
 DOM Tree   

 Lists: 

 Unordered List (without numbering) - <ul> & <li>
 Ordered List (numbers) - <ol> & <li>
 <li> - use to wrap every element inside List

Parent tags or outer tags: ol, ul, dl
inner tags: li, dt ,dd

Description lists:
dl - desciption list
dt - define term (your word or sentence which needs to be described eg. course name)
dd - describe (write your description for the above term or word)
